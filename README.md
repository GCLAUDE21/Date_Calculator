# DATE CALCULATOR

Projet réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/Date_Calculator/)

## Objectif

Créer un ensemble de fonctions utilitaires pour la gestion des champs de date en JavaScript natif.
L'utilisateur peut configurer un formulaire de réservation : la date de début est fixée à aujourd'hui,
la date de fin à demain, et changer la date de début met à jour automatiquement la date de fin.

## Compétences travaillées

- Manipulation native des dates JavaScript (`Date`, `toISOString`, `Date.parse`)
- Interaction avec les inputs HTML de type date
- Gestion des événements DOM (`addEventListener`, `change`)
- Validation et synchronisation de formulaire
- Refactorisation de code

## Stack technique

JavaScript natif. Aucun framework ou bibliothèque externe.
